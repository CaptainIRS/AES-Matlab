ShiftRows() Transformation

In the ShiftRows() transformation, the bytes in the last three rows of the State are cyclically shifted over different numbers of bytes (offsets). The first row, r = 0, is not shifted.

This has the effect of moving bytes to “lower” positions in the row (i.e., lower values of c in a given row), while the “lowest” bytes wrap around into the “top” of the row (i.e., higher values ofc in a given row).

function [state] = ShiftRows(state)
    for i = 2:4
        state(i, :) = circshift(state(i, :), 1-i);
    end
end
